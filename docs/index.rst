edeposit.amqp.ftp
=================

This module provides wrappers over ProFTPD_ for edeposit_ project. It allows
automatic and/or batch uploads of both files and metadata.



.. _ProFTPD: http://www.proftpd.org/
.. _edeposit: http://edeposit.nkp.cz/
